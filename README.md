# 🏥 Klinik Sederhana

Sistem Informasi Klinik Sederhana merupakan proyek mata kuliah **Pemrograman Berorientasi Objek (PBO)** yang dikembangkan oleh **Kelompok 1**. Aplikasi ini bertujuan untuk membantu pengelolaan data klinik seperti data pasien, dokter, obat, dan pemeriksaan secara lebih terstruktur dan efisien.

---

## 📖 Deskripsi Proyek

Aplikasi Klinik Sederhana dibuat menggunakan konsep **Object-Oriented Programming (OOP)** dengan bahasa pemrograman Java. Sistem ini dirancang untuk mempermudah proses administrasi klinik melalui fitur pengelolaan data yang terintegrasi dengan database.

---

## ✨ Fitur Utama

- 👨‍⚕️ Manajemen Data Dokter
- 🧑‍🤝‍🧑 Manajemen Data Pasien
- 💊 Manajemen Data Obat
- 🩺 Manajemen Data Pemeriksaan
- 📋 Riwayat Pemeriksaan Pasien
- 🗄️ Integrasi Database MySQL
- 🖥️ Antarmuka Desktop Menggunakan Java Swing

---

## 🛠️ Teknologi yang Digunakan

- Java
- Java Swing
- MySQL
- JDBC
- NetBeans IDE

## 🚀 Cara Menjalankan Program

1. Clone repository

```bash
git clone https://github.com/Alfanwahyudicode/klinikPratama.git
```

2. Import project ke NetBeans.
3. Buat database MySQL sesuai file SQL yang disediakan.
4. Sesuaikan konfigurasi koneksi database.
5. Jalankan program melalui NetBeans.
